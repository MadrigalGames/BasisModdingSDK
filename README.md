# Basis modding SDK

## ❔ What's this?

This repository contains the code for the **Basis Game Engine's** modding SDK. Basis is Madrigal Ltd's game engine, used for games such as **[Traction Point](https://store.steampowered.com/app/1485840/Traction_Point/)**.

## 🚀 Getting Started

Check out the [Traction Point modding](https://github.com/MadrigalGames/TractionPointModding) repository for documentation and example mods.
